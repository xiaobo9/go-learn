# golang learn

## 初始化项目

```bash
go mod init github.com/xiaobo9/a-go-learn
go get # 引入
go install # 安装编译
```

## 开发运行

```bash
go get github.com/lxn/walk

go get github.com/akavel/rsrc

rsrc -manifest main.exe.manifest -o rsrc.syso

go build

go build -ldflags="-H windowsgui"
```
